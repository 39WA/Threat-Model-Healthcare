# Healthcare System Threat Model

## Overview

This repository contains a comprehensive **threat model of a healthcare application**, developed using industry-standard threat modeling methodologies. The project identifies key system components, trust boundaries, potential threats, and recommended security mitigations to reduce organizational risk.

This work demonstrates practical skills in **secure architecture analysis, threat identification, and risk assessment**, aligned with real-world cybersecurity practices.

---

## Objectives

- Identify critical assets within a healthcare system
- Analyze the system architecture and data flows
- Identify potential threats using structured threat modeling techniques
- Assess risk severity and likelihood
- Recommend security controls and mitigations
